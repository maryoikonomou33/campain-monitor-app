Download and extract the project folder, then open a terminal or command prompt and navigate to the folder using cd path/to/project-folder. Create and activate a virtual environment with python -m venv venv and venv\Scripts\Activate on Windows or python3 -m venv venv and source venv/bin/activate on Mac/Linux. Install dependencies by running pip install -r requirements.txt. Start the Flask server with python app.py and open http://127.0.0.1:5000 in your browser. To add a subscriber, enter a name and email in the form and click "Add Subscriber". To remove a subscriber, click "Remove" next to their name.
